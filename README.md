## Installation

1. Clone repo

  ```bash
  git clone https://github.com/code-kotis/qr-code-scanner
  ```

2. Installation

  ```bash
  npm install
  ```

3. Run

  ```bash
  npm run start
  ```

4. Build

  ```bash
  npm run build
  ```
